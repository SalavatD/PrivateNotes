# PrivateNotes
Temporary notes app.

## Running
1. Clone this repository:
```
git clone https://github.com/SalavatD/PrivateNotes.git
```
2. Go to directory:
```
cd PrivateNotes
```
3. Build Docker image:
```
docker build -t privatenotes .
```
4. Run container:
```
docker run -p 80:8080 privatenotes
```
5. Go to [localhost](http://localhost).
